# Huaxia: Warring States Glossary

## 사용 원칙

- 이 문서는 번역 전 우선 확인용 사전이다.
- 자동 부분 치환용 파일이 아니다.
- 같은 한자라도 문맥에 따라 다를 수 있으므로, 최종 문장은 문맥을 우선한다.
- 원문에 없는 괄호 설명이나 한자 병기를 추가하지 않는다.
  - 예: `상선약수(上善若水)`처럼 원문에 없는 한자 병기 금지.
  - 예외: 원문 자체에 괄호나 병기가 있을 때만 대응 번역에 반영.
- `精 / 气 / 神`은 괄호 한자 병기를 쓰지 않는다.
  - `精=정`
  - `气=기`
  - `神=신`
- `气`는 문맥에 따라 `기`, `기운`, `진기`, `영기`, `기해` 등으로 번역한다.
- 인명, 지명, 세력명은 로컬 Qwen 번역 후 사람 검수 우선.
- 단일 한자 용어는 자동 부분 치환하지 않는다.
  - 예: `子=자`는 원문 전체가 `子`일 때만 적용.
  - 예: `子正`을 `자正`처럼 중간 글자만 치환하지 않는다.
  - 성씨, 국가 단자, 지도 단자, 시각 단자는 문자열 전체 또는 확정 복합어 기준으로만 처리한다.

## 번역 금지 또는 주의

- 원문에 없는 괄호 한자 병기 추가 금지.
- 사자성어, 고사성어, 무협 용어도 한국어 번역만 사용한다.
  - 예: `운기조식`, `상선약수`, `천의무봉`처럼 표기.
  - 원문에 한자 병기가 없으면 `(運氣調息)` 같은 설명을 붙이지 않는다.
- `<color...>`, `</color>`, `<link...>`, `<size...>` 등 태그는 보존.
- `{0}`, `{1}`, `%s`, `{{A}}`, `{ref=...}` 변수는 보존.
- 숫자, 배율, 퍼센트, 단위는 임의 삭제 금지.
- 지도 위 단일 한자 라벨은 원문 유지 가능.
- `remark`는 대부분 개발자 메모다. 실제 출력 여부 확인 전 번역하지 않는다.
