# 화하사시: 전국 (HuaXia: Warring States) 번역 톤 가이드

## 1. 게임 세계관 개요

* **시대적 배경:** 주나라 말기, 혼란스러운 춘추전국시대(春秋战国时代).
* **주요 테마:** 노예 신분에서 시작하여 세력을 규합하고 천하를 제패하거나, 홀로 구주(九州)를 유랑하는 무협/선협적 영웅 서사.
* **핵심 요소:** 
  * 수신제가치국평천하(修身齊家治國平天下)를 거쳐 제왕업(帝王業)을 이루거나 만세에 이름을 남기는 서사.
  * 문파/씨족 창설, 세력 경영, 군대 편제 및 원정, 합종연횡(合縱連橫) 등 SLG 요소.
  * 신마난무(神魔亂舞)의 신화적 배경 속에서 정·기·신을 수련하며 중원(中原)을 제패하는 여정.

## 2. 번역 톤앤매너 (Tone & Manner)

* **기본 문체:** 가벼운 현대어보다는 고풍스럽고 무게감 있는 **문어체**와 **사극/무협풍 어조**를 지향한다.
* **대화문:** 
  * 신분과 위계(사부/제자, 군주/신하)에 따른 존대와 하대를 명확히 구분한다.
  * 과도한 인터넷 유행어나 현대적인 조어는 배제한다.
* **설명문:** 시스템이나 스킬 설명은 분위기보다 **정확성과 가독성**을 최우선으로 하되, 용어 자체는 무협풍을 유지한다. (예: HP -> 기혈, MP/마나 -> 진기/내력)
* **괄호 병기 금지:** 원문에 없는 한자 병기나 해설 괄호를 추가하지 않는다. 사자성어와 무협 용어도 한국어 표기만 사용한다.

## 3. 무협/선협 특화 용어 처리 가이드

한국의 무협/선협 독자 및 게이머들에게 익숙한 한자어 위주로 번역하여 장르적 몰입감을 극대화한다.

* **스탯 및 능력치:**
  * 体魄 (체백) -> **근골 (筋骨)**
  * 力量 (역량) -> **근력 (筋力)**
  * 敏捷 (민첩) -> **신법 (身法)**
  * 气海 (기해) -> **기해** (단전/내공의 그릇)
  * 灵犀 (영서) -> **오성 (悟性)** (깨달음의 재능)
  * 感知 (감지) -> **영각 (靈覺)** (기운을 감지하는 육감)
  * 才智 (재지) -> **자질 (資質)** (경험치 획득 및 수련 재능)
  * 权谋 (권모) -> **모략 (謀略)**
  * 武略 (무략) -> **통솔 (統率)** (부대 통솔 및 사기 관리)

* **핵심 수련 개념 (정/기/신):**
  * UI 및 좁은 라벨: **정, 기, 신**
  * 일반 설명문: **정, 기, 신**

* **관계 및 세력:**
  * 江湖 -> **강호**
  * 门派 -> **문파**
  * 师傅 / 徒弟 -> **사부 / 제자**
  * 师兄 / 师弟 -> **사형 / 사제**

* **효과 및 특성명:**
  * 4글자 한자성어 형태의 직관적인 작명을 우선한다.
  * `~血脉` -> `~혈맥` (예: 기린혈맥, 백택혈맥 - 띄어쓰기 없이 붙여 씀)
  * 출생 월/계절 효과는 예외적으로 자연스러운 한국어 표기 허용 (예: 초봄 출생, 연주태생).

## 4. UI 및 시스템 텍스트 처리 규칙 (기술적 제약)

* **텍스트 오버플로우 방지:** 무협 용어는 한자 압축률이 높아 한국어로 풀어쓰면 UI를 벗어나는 경우가 많다. 좁은 버튼이나 라벨에 들어가는 텍스트는 2~4글자 한자어풍으로 압축한다.
* **지도 라벨의 통일성:** 지명(예: 郡)을 나타내는 단일 한자는 원문을 유지하여, 한자와 한글 단위가 어색하게 섞이는 것을 방지한다.
* **변수 및 태그 보존:** `{0}`, `%s`, `<color=#FFFF00>` 등의 시스템 변수와 리치 텍스트 태그는 원문과 100% 동일하게 위치시켜야 한다.
* **부분 치환 금지:** 고유명사가 오염되는 것을 막기 위해 (예: '혈맥' 부분 치환 시 '기린혈맥' 전체 매칭 실패), 검수 단계에서는 자동 치환(Substitutions) 대신 통문장 번역을 원칙으로 한다.
